## Running Game

Clone the repository

    git clone https://github.com/lawrencehayford/Ceros--Game.git

Switch to the repo folder

    cd Ceros--Game

Install all dependencies

    npm install

Start Game

    npm start

Browse http://localhost:5000 to view Game

Run Test Using

    npm run test
